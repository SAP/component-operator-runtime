  Bootstrap a new component operator

There are two supported ways to bootstrap a new component operator based on
component-operator-runtime:

- **[Using the scaffolder](scaffolder/)** — generate a ready-to-use project skeleton
  with the provided scaffolding tool.
- **[Using kubebuilder](kubebuilder/)** — integrate the framework into a project
  managed by [kubebuilder](https://book.kubebuilder.io/).

Pick whichever fits your workflow and follow the corresponding guide.
